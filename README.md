# R package: affxparser


## Installation
Package is available on [Bioconductor](http://www.bioconductor.org/packages/devel/bioc/html/affxparser.html).  Install in R as:

```r
source('http://bioconductor.org/biocLite.R')
biocLite('affxparser')
```
